

open app

select a drive and click process.

Then appp wil create a .ini file with the log in it.

upon giving instructions to proceed, it will start copy and keep a track of the process.

the if card content is not deleted, then .ini file will stay. if the data is moved, ini will also gets deleted.



Ultimate log file will be a csv file which will keep track of every transaction.
